# Loan EMI Calculator

A responsive Loan EMI Calculator built using Flask, HTML, and CSS.

## Features
- EMI calculation with validation
- Currency selection
- Mobile responsive UI

## Tech Stack
- Python (Flask)
- HTML
- CSS
- JavaScript

## How to Run
1. Install Flask
2. Run `python app.py`
3. Open browser at `http://127.0.0.1:5000/`
